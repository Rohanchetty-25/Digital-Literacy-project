## Key Learnings & Reflections

### Top 5 Takeaways

1. **Digital identity is permanent** — Every profile, post, and interaction contributes to my professional reputation
2. **Communication clarity matters** — Professional emails require structure, not length
3. **Security is proactive** — 2FA and password managers are essential, not optional
4. **Tools build skills** — Platforms like HackerRank and GitHub are practice grounds for real-world work
5. **Awareness protects** — Understanding phishing tactics is the first line of defense
